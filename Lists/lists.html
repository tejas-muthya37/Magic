Lists

Lists are usually used to show multiple actions to the user at once. Use the .list class to add our lists to your website.

Simple List

Simple Lists are mostly used to list different primary actions.

Mailbox

  • Inbox
  • Starred
  • Drafts 
              
<div class="list">
  <header>
    <h3>Mailbox</h3>
  </header>
  <div class="nav-items">
    <ul>
    <li>
      <a href="#">Inbox</a>
    </li>
    <li>
      <a href="#">Starred</a>
    </li>
    <li>
      <a href="#">Drafts</a>
    </li>
    </ul>
  </div>
</div>

Nested List

Nested Lists are mostly used to list different primary and secondary actions.

Mailbox

  • Inbox
  • Starred
  • Sent
  • Today
  • This Week
  • This Month
  • Drafts 
              
<div class="list">
  <header>
    <h3>Mailbox</h3>
  </header>
  <div class="nav-items">
    <ul>
    <li>
      <a href="#">Inbox</a>
    </li>
    <li>
      <a href="#">Starred</a>
    </li>
    <li>
      <a href="#">Sent</a>
      <li class="nested-nav-item">
                Today</li>
      <li class="nested-nav-item">
                This Week</li>
      <li class="nested-nav-item">
                This Month</li>
    </li>
    <li>
      <a href="#">Drafts</a>
    </li>
    </ul>
  </div>
</div>

Select List

Select Lists can be used to implement filters in an E-Commerce App.

Filters

              
<div class="list">
  <header>
      <h3>Filters</h3>
  </header>
  <div class="nav-items">
      <ul>
      <li>
        <input type="checkbox" checked>
        <label>Thrillers</label>
      </li>
      <li>
        <input type="checkbox">
        <label>Romance</label>
      </li>
      <li>
        <input type="checkbox">
        <label>Drama</label>
      </li>
      </ul>
  </div>
</div>

Stacked List

Stacked Lists can be used to show notifications on an App.

Notifications

  • Message from Tejas (1)
  • Message from Tanay (3)
  • Manish liked your post

      <h3>Notifications</h3>
  </header>

      <ul>
