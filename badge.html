Badges

Badges are used to add additional information to any content. Use the .badge class to add badges to your website.

Badges On Text Elements

There are 2 types of badges for text elements. Small and a Large badge.

Text with a badge

4

Text with a large badge

1 
              
<div class="badge">
    <p>Text with a badge</p>
    <span>4</span>
</div>
  
<div class="badge badge-lg">
    <p>Text with a large badge</p>
    <span>1</span>
</div>

Badges On Buttons

There are 2 types of badges for buttons. Normal and a Toggling badge.

8
7 
              
<div class="badge badge-left">
    <button>Action</button>
    <span>8</span>
</div>
  
<div class="badge badge-toggle">
    <button>Hide</button>
    <span>7</span>
</div>
<!-- Add the below code to your .js file or <script> tag for the toggle to work -->
<script>
    const toggleButton = document.querySelector(".badge-toggle > button");
    const toggleBadge = document.querySelector(".badge-toggle span");
    toggleButton.addEventListener("click", function () {
      toggleBadge.classList.toggle("hidden");
    });
</script>

Badges On Icons

Badges On Avatars

Badges on avatars can be used to show the status of a user.
