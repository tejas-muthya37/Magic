Navigation

Navigation bars are used to help the user navigate to the different end points of the website. Use the .navbar class to use our Navbars in your website.

Simple Navbar

Simple Navbars have a logo and links to different endpoints.

Magic

              
<nav class="navbar">
  <div class="navbar-left">
    <i class="fas fa-bars fa-lg"></i>
    <h3>Magic</h3>
  </div>
  <div class="navbar-right">
    <i class="fas fa-download fa-lg"></i>
    <i class="fas fa-print print-icon fa-lg"></i>
    <i class="far fa-bookmark fa-lg"></i>
  </div>
</nav>

Prominent Navbar

Prominent Navbars have the same content as Simple Navbars but look more prominent.

Magic

              
<nav class="navbar prominent-dense">
  <div class="navbar-left">
    <i class="fas fa-bars fa-lg"></i>
    <h3>Magic</h3>
  </div>
  <div class="navbar-right">
    <i class="fas fa-download fa-lg"></i>
    <i class="fas fa-print print-icon fa-lg"></i>
    <i class="far fa-bookmark fa-lg"></i>
  </div>
</nav>

Collapsed Navbar

Collapsed Navbars are generally used when the Navbars need to be given less prominence. 

              
            <nav class="navbar collapsed">
              <div class="navbar-left">
                <i class="fas fa-bars fa-lg"></i>
              </div>
              <div class="navbar-right">
                <i class="far fa-bookmark fa-lg"></i>
              </div>
            </nav>
