Buttons

Buttons. Need we say anything more? Use the .btn class to add our buttons to your website.

Primary Buttons

Primary Buttons are used to draw attention to your Primary CTAs. 

              
<button class="btn btn-primary">Primary</button>

<button class="btn btn-primary btn-outline">Primary</button>

Link Buttons

The purpose of a Link Button is to navigate the user to the respective endpoint. 

              
<button class="btn btn-link">
    <a href="#">Link</a>
</button>
            
<button class="btn btn-link btn-outline">
    <a href="#">Link</a>
</button>

Icon Buttons

Buttons with icons can be used to convey what the button does.

              
<button class="btn btn-icon">
    <i class="fa fa-leaf" aria-hidden="true"></i> Icon
</button>
            
<button class="btn btn-icon btn-outline">
    <i class="fa fa-leaf" aria-hidden="true"></i> Icon
</button>

Floating Action Button

A floating action button is used to perform a specific action across the webpage. 

              
<button class="btn-floating-action">+</button>
