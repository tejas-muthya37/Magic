Inputs

Inputs are the way for your users to communicate with the website.

Login Inputs

Login inputs have a username and password field. We have designed an error message as well which you can display accordingly.

Wrong Password. Try again.
              
<div class="login-inputs">
  <div>
      <label for="email">Username *</label>
      <input type="email" id="email" />
  </div>
  <div>
      <label id="password-label" for="password">Password *</label>
      <input type="password" id="password" />
      <div class="error-message">Wrong Password. Try again.</div>
  </div>
</div>

Radio Inputs

Radio inputs are used when there is only one possible answer to a question. 

              
<div class="radio-input-group">
  <div>
    <label for="checked-radio">Checked</label
    ><input checked id="checked-radio" type="radio" />
  </div>
  <div>
    <label for="unchecked-radio">Unchecked</label
    ><input id="unchecked-radio" type="radio" />
  </div>
  <div>
    <label for="disabled-radio">Disabled</label
    ><input disabled id="disabled-radio" type="radio" />
  </div>
</div>

Checkbox Inputs
