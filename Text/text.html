Text Utilities

Can't do without showing text to your user, can you? Use the .text-utilities class to add our text utilities to your website.

Headings

Headings often signify the start of a particular section.

h1. This is a <h1/> heading.

h2. This is a <h2/> heading.

h3. This is a <h3/> heading.

h4. This is a <h4/> heading.

h5. This is a <h5/> heading.
h6. This is a <h6/> heading. 
              
<h1 class="text-utilities heading">
    h1. This is a <h1/> heading.
</h1>

<h2 class="text-utilities heading">
    h2. This is a <h2/> heading.
</h2>

<h3 class="text-utilities heading">
    h3. This is a <h3/> heading.
</h3>

Small Text

Small Text is used when there are multiple lines of information.

This is an example of a small text.

              
<p class="text-utilities">This is an example of a small text.</p>

Centered Text

Centered Text is always placed at the exact center of the screen.

This is an example of a centered text.

Centered text.

              
<p class="text-utilities centered">This is an example of a centered text.</p>

Grey Text

Grey Text usually signifies something that is disabled.

This is an example of a grey text. 

              
<p class="text-utilities grey">This is an example of a grey text.</p>
