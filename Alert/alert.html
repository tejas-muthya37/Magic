Alerts

Alerts, as the name suggest are used to alert the user about an event. Use the .alert class to add our alerts to your website.

Simple Alerts

Simple alerts are of 4 types. Primary, Success, Warning and Danger.

Primary Alert! This is a Primary Alert.
Success Alert! This is a Success Alert.
Warning Alert! This is a Warning Alert.
Danger Alert! This is a Danger Alert. 
              
<div class="alert primary">
  <span>Primary Alert!</span> This is a Primary Alert.
</div>
<div class="alert success">
  <span>Success Alert!</span> This is a Success Alert.
</div>
<div class="alert warning">
  <span>Warning Alert!</span> This is a Warning Alert.
</div>
<div class="alert danger">
  <span>Danger Alert!</span> This is a Danger Alert.
</div>
